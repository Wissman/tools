
#################################################
#               DOWNLOAD TOOLKIT                #
#################################################
open terminal (/home/username) and write:
git clone git://git.code.sf.net/p/netoolsh/opensource-kali opensource



#################################################
#               INSTALL TOOLKIT                 #
#################################################

Just browse to opensource folder in our terminal and
execute: chmod +x INSTALL.sh follow by: ./INSTALL.sh

example:
cd opensource
chmod +x INSTALL.sh
./INSTALL.sh
